# 九宮奇門鎖單宮

V4.3 靜態版 App。這版以「自用與分享給朋友」為優先，介面保持簡約：輸入時間、排盤、選 1-9、看結果、存案例。

## 功能

- 輸入西元時間，自動換算：農曆、四柱、陰盤局數、旬首、符頭、空亡、驛馬、值符、值使。
- 支援 1-9 報數鎖單宮；使用者介面只顯示可選數字與鎖定結果，不額外介紹對應規則。
- 生成九宮盤面：八神、九星、八門、天盤干、地盤干、空亡、驛馬、入墓、擊刑、門迫、門制。
- 紅吉綠凶符號標示，並加上「吉 / 凶 / 中」文字標籤，避免只靠顏色判讀。
- 手機版先顯示鎖單宮結果、行動建議與風水改善，再往下看九宮盤。
- 自動產生總斷、行動建議、風水改善方法。
- 報告可切換「簡明 / 詳細」兩種閱讀深度。
- 支援列印/PDF、複製報告、下載文字、匯出 / 匯入盤面 JSON。
- 內建案例庫，可儲存本次盤、補填實際結果、搜尋、回看、刪除、匯出案例 JSON。
- 已拆分為 `index.html`、`style.css`、`engine.js`、`app.js`，並加入 PWA `manifest.webmanifest` 與 `sw.js`。
- 規則資料放在 `rules/`，包含鎖宮、分數、用途權重資料位與風水建議守則。

## 起局規則

陰盤九宮局數：

> 年支數 + 農曆月 + 農曆日 + 時支數，除 9 取餘數，餘 0 作 9。

例：2026/07/08 07:44，農曆五月二十四，丙午年、丙辰時：

> 午 7 + 5 + 24 + 辰 5 = 41，41 % 9 = 5，所以陰五局。

## 注意

此版本使用公開資料與使用者提供樣本盤面校準。底層保留報數鎖宮表、直接否定法、20 / 20 / 40 / 10 / 10 加分規則，但使用者介面不主動介紹未必要知道的鎖宮轉換細節。

待確認：

- 目前起局仍沿用 v3 原本的陰盤九宮公式；若要完全對齊其他起盤體系，需再補正式起盤規則或更多樣本盤。
- 用途選單目前主要用於顯示與報告語境，`rules/qtype-rules.json` 已建立資料位，但標註「待確認」，尚未套用「財運 / 工作 / 感情」各自不同的正式權重。
