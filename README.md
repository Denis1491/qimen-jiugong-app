# 九宮奇門鎖單宮｜全自動排盤系統 V3

單檔離線版 HTML App。

## 功能

- 輸入西元時間，自動換算：農曆、四柱、陰盤局數、旬首、符頭、空亡、驛馬、值符、值使。
- 支援 1-9 報數鎖單宮：1 坎、2 坤、3 震、4 巽、5 改取坤二宮、6 乾、7 兌、8 艮、9 離。
- 生成九宮盤面：八神、九星、八門、天盤干、地盤干、空亡、驛馬、入墓、擊刑、門迫、門制。
- 紅吉綠凶符號標示：大凶符號直接否定，無大凶時再依 PDF 權重計分。
- 自動產生總斷、行動建議、風水改善方法。
- 支援列印/PDF、複製報告、下載文字、匯出 JSON。

## 起局規則

陰盤九宮局數：

> 年支數 + 農曆月 + 農曆日 + 時支數，除 9 取餘數，餘 0 作 9。

例：2026/07/08 07:44，農曆五月二十四，丙午年、丙辰時：

> 午 7 + 5 + 24 + 辰 5 = 41，41 % 9 = 5，所以陰五局。

## 注意

此版本使用公開資料與使用者提供樣本盤面校準。已套用報數 5 改取坤二宮、直接否定法、20 / 20 / 40 / 10 / 10 加分規則。

待確認：目前起局仍沿用 v3 原本的陰盤九宮公式；若要完全對齊其他起盤體系，需再補正式起盤規則或更多樣本盤。

農曆換算支援 1900-2050。
