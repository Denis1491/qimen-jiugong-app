# 九宮奇門極簡版

V5.4 將朋友版介面收斂成一條最短路徑：

1. 輸入一件具體事情。
2. 選擇問事類型。
3. 憑直覺選 1–9。
4. 查看分數與「偏吉／偏凶」。

## 畫面原則

- 起盤時間預設為現在，需要時才展開修改。
- 結果頁只顯示問題、題型、所選數字、分數、吉凶、一句安全提醒與一個重新開始按鈕。
- 不顯示九宮盤、宮位轉換、術語、推導、報告、案例、分享或校準功能。
- 舊案例資料與 `qimen-jiugong-cases-v5` 儲存鍵均保留，不會因介面簡化而刪除。

## 計算範圍

本版只簡化介面，沒有重寫起盤、鎖宮或計分引擎。底層仍會完整生成九宮資料，再依既有 V5.0 暫行規則計分。

- 60 分以上：偏吉。
- 未滿 60 分：偏凶。
- 嚴重風險組合：凶，不宜強推。

術法規則與題型權重仍標示「待校準」。分數只代表目前規則下的傾向，不是成功率、準確率，也不構成健康、法律、財務或投資保證。
