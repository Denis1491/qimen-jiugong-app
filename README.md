# 九宮奇門極簡版

V5.5 保留朋友版最短路徑，底層改為可重現的時家奇門排盤：

1. 輸入一件具體事情。
2. 選擇問事類型。
3. 憑直覺選 1–9。
4. 查看分數與「吉／凶」。
5. 需要核對時，再進入完整九宮盤。

## 畫面原則

- 起盤時間預設為現在，需要時才展開修改。
- 結果頁先顯示問題、題型、所選數字、分數、吉凶與一句安全提醒。
- 「查看完整盤面」會開啟獨立九宮頁；主格顯示宮位、八神、九星、八門，點宮後可看天地盤干與特殊象。
- 完整盤面不顯示各宮分數或吉凶底色，避免分數干擾原始盤面閱讀。
- 不顯示長篇報告、案例、分享或校準功能。
- 舊案例資料與 `qimen-jiugong-cases-v5` 儲存鍵均保留，不會因介面簡化而刪除。

## 計算範圍

底層採用以下固定設定，讓同一時間永遠得到同一張盤：

- 陽盤時家奇門。
- 超接置閏法，九日門檻。
- 轉盤法。
- `Asia/Taipei`（UTC+8）民用時間，不校正真太陽時。
- 子正換日（00:00）。
- 天禽與中宮干寄坤，隨天芮轉動。
- 八神顯示為：值符、螣蛇、太陰、六合、白虎、玄武、九地、九天。

分數只讀報數鎖定的單宮，不使用題型權重、亂數或日期雜湊：

- 白虎、天蓬、天芮、死門、庚或空亡，任一命中即直接判凶。
- 無直接否定時：吉神 +20、吉星 +20、吉門 +40、天盤吉干 +10、地盤吉干 +10。
- 坤二宮會同時讀取本宮地盤干與中宮寄干；同一層吉干仍只加一次 10 分，寄入的庚仍直接判凶。
- 60 分以上：吉。
- 未滿 60 分：凶。
- 報數 5 固定改取坤二宮。

分數是書中規則的符號加總，不是成功率或科學機率，也不構成健康、法律、財務或投資保證。節氣採離線太陽視黃經算法；香港天文台 2026 年 24 個節氣均通過一分鐘內回歸。由於輸入只到分鐘，若時間正好落在交節同一分鐘，仍建議另盤核對。

## 規則與驗收來源

- 子奇老師《九宮奇門》：[出版社書頁](https://www.cite.com.tw/book?id=88010)、[Google Books 操作原文](https://books.google.com/books?id=YJwtEAAAQBAJ&pg=PT24&hl=zh-TW)。
- 超神、接氣、置閏、旬首與轉盤：[《奇門遁甲統宗》卷一](https://ctext.org/wiki.pl?chapter=666094&if=gb)、[卷二](https://ctext.org/wiki.pl?chapter=491157&if=gb)。
- 完整轉盤案例：[《武備志》卷 176–177](https://ctext.org/wiki.pl?chapter=1324023&if=gb)。
- 節氣核對：[香港天文台二十四節氣資料](https://www.hko.gov.hk/tc/gts/astronomy/Solar_Term.htm)。
- 回歸測試：書中 2019、2020 案例，以及 2009、2013 公開完整盤。

node --check app.js
